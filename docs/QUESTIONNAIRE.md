# StayWithMeForever · 问卷题库设计

## 设计原则

1. **行为投射**：不问"你外向吗？"，而是问"聚会上你通常会..."（行为场景）
2. **情境化**：每道题落在一个具体场景，避免抽象自评
3. **排序与权衡**：价值观部分用"不得不放弃哪个"来迫使用户排序
4. **防社会期许**：选项设计让每个选项都有合理的动机

## 模块结构

| 模块 | 缩写 | 题数 | 类型 | 提取目标 |
|------|------|------|------|---------|
| 基础画像 | P0 | 10 | 自填 + 单选 | MBTI 自评、Big Five 自评、年龄范围、职业等 |
| 认知模式 | P1 | 25 | 情境单选 + 排序 | 决策风格、信息处理方式、问题解决路径 |
| 价值观 | P2 | 20 | 两难单选题 + 排序 | 自由/安全、成就/关系、长期/短期 |
| 表达与情感 | P3 | 25 | 风格单选题 | 情绪反应、沟通风格、幽默偏好、确定性表达 |
| 社交与关系 | P4 | 20 | 角色单选题 | 对不同关系角色（友/恋/家/同事）的行为模式 |
| 自由表达 | P5 | 6 | 开放简答 | 表达 DNA 原文采集 |

**总计**：约 106 题。预计答题时间：15-20 分钟。

## 各模块详解

### P0：基础画像（10题）
用户可直接输入已知 MBTI 类型，无需重新测试。如果不知道，可通过后续选择题间接推导。

题目覆盖：
- 名字 / 称呼
- MBTI 自评（支持"不太清楚"跳过）
- 年龄范围
- 职业领域
- 内向/外向自评
- 逻辑/感受自评
- 开放/保守自评
- 共情能力自评
- 自律/条理自评
- 压力应对方式

### P1：认知模式（25题）
重点提取：分析型 vs 直觉型、框架型 vs 碎片型、冒险型 vs 保守型。

代表性题目：
- P1-001：面对复杂未知问题，第一步做什么？→ 决策起始风格
- P1-003：学习新技能的方式 → 学习风格
- P1-005：思维结构（地图/拼图/树状/螺旋）→ 认知结构
- P1-006：面对科学研究报道的第一反应 → 批判性思维
- P1-009：内心的思维方式（语言/图像/感觉）→ 思维媒介
- P1-024：3个月自由时间的用途 → 内在驱动力

### P2：价值观（20题）
重点提取：价值优先级排序、道德两难偏好、对金钱/名望/原谅的态度。

代表性题目：
- P2-001：六项价值观排序（自由/安全/成长/关系/意义/认可）
- P2-002：朋友创业失败借钱 → 利他 vs 边界
- P2-005："宁可放过一千" vs "付出该得回报" → 正义观
- P2-007：必须在名声和财富间放弃一个 → 社会价值观
- P2-017：选合伙人"聪明的混蛋"vs"善良的平庸"→ 人际价值权衡

### P3：表达与情感（25题）
重点提取：沟通方式、情绪表达、幽默类型、文字风格。

代表性题目：
- P3-003：说话风格（直接/铺垫/叙事/启发）→ 表达节奏
- P3-004：句子长度偏好 → 句式特征
- P3-005：生气时的表达方式 → 情绪外显模式
- P3-015："我觉得"vs"我认为"→ 主语倾向
- P3-018：更喜欢的赞美类型 → 自我确认需求
- P3-020：动词/名词/形容词偏好 → 词性风格

### P4：社交与关系（20题）
重点提取：不同关系的互动模式、社交能量、边界处理。

代表性题目：
- P4-002：陌生人社交后的能量状态 → 社交电池
- P4-004：伴侣关系中矛盾处理方式 → 冲突解决风格
- P4-009：聊天中是说话者还是倾听者 → 社交角色
- P4-012：给朋友建议的风格 → 助人模式
- P4-017：恋爱中粘人 vs 需要空间 → 依恋风格

### P5：自由表达（6题）
这些题目是整个 Skill 质量的关键，直接提供"表达 DNA"的原材料。

六大场景：
1. **安慰朋友失恋** → 同理心模式
2. **拒绝同事请求** → 边界与拒绝风格
3. **解释电脑变慢** → 复杂概念降维能力
4. **理想周末描述** → 自然状态下的语言习惯
5. **反驳不认同观点** → 论证与反驳风格
6. **审视自身变化** → 自我认知深度与时间观

## 简答题的 AI 分析重点

每个简答题的原始文本，LLM 会重点分析：
- 句子长度与复杂度
- 词汇情感色彩（积极/消极/中性）
- 逻辑结构（因果/并列/转折/递进）
- 人称使用（我/你/我们）
- 语气词和辅助词（吧/嘛/呢/了/的）
- 比喻和修辞使用
- 回避或直面程度
